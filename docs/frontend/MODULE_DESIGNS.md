## 1. 系统管理模块

### 1.1 用户管理页面

**路由**: `/system/user`  
**布局**: 左右分栏(左侧部门树 + 右侧用户列表)

#### 页面结构
```
┌──────────────────────────────────────────────────────┐
│ 搜索表单                                              │
│ [用户名____] [手机____] [状态▼] [🔍搜索] [↻重置]      │
├──────────┬───────────────────────────────────────────┤
│          │ 工具栏                                     │
│ 部门树   │ [+新增] [批量删除] [导入] [导出]           │
│          ├───────────────────────────────────────────┤
│ 总公司   │ 用户列表表格                               │
│ ├销售部  │ ☑ | 用户名 | 昵称 | 手机 | 状态 | 操作    │
│ ├市场部  │ ☐ | admin  | 管理员|138* | ✅  | 编辑|删除│
│ └财务部  │ ☐ | zhangsan|张三 |139* | ⛔  | 编辑|删除│
│          ├───────────────────────────────────────────┤
│          │ 分页: 共100条 < 1 2 3 ... > 跳至[__]页     │
└──────────┴───────────────────────────────────────────┘
```

#### 交互设计
- **左侧部门树**: 点击部门节点,右侧自动筛选该部门用户
- **新增按钮**: 弹出抽屉式表单(Drawer),宽度600px
- **编辑按钮**: 同样使用抽屉,回填数据
- **删除**: 二次确认弹窗,"确定要删除用户 xxx 吗?"
- **批量操作**: 勾选后工具栏显示"已选择x项"

#### 表单字段(新增/编辑)
```typescript
{
  username: string        // 用户名(必填,唯一)
  nickname: string        // 昵称(必填)
  deptId: number          // 所属部门(树形选择器)
  email: string           // 邮箱(必填,格式校验)
  phone: string           // 手机号(必填,格式校验)
  gender: '0' | '1' | '2' // 性别(单选)
  roleIds: number[]       // 角色(多选下拉框)
  status: '0' | '1'       // 状态(开关)
  remark: string          // 备注(可选)
}
```

### 1.2 角色管理页面

**路由**: `/system/role`  
**布局**: 标准列表页

#### 特色功能
- **数据权限设置**: 点击"数据权限"按钮,弹出对话框
  - 单选: 全部数据 / 本部门 / 本部门及以下 / 仅本人 / 自定义

- **分配菜单**: 右侧滑出抽屉,显示树形菜单,可勾选

#### 视觉设计
- 角色卡片式展示(而非表格),更直观

### 1.3 菜单管理页面

**路由**: `/system/menu`  
**布局**: 树形表格

#### 交互亮点
- **拖拽排序**: 支持拖拽调整菜单顺序
- **图标选择器**: 点击图标字段,弹出Element Plus图标网格,可搜索
- **展开/折叠**: 默认展开所有层级

#### 表单特殊字段
```typescript
{
  menuType: 'M' | 'C' | 'F'  // 菜单类型(单选: 目录/菜单/按钮)
  icon: string                // 图标(图标选择器)
  path: string                // 路由地址
  component: string           // 组件路径(仅菜单类型)
  perms: string               // 权限标识(如 customer:add)
  visible: boolean            // 是否显示(开关)
}
```

## 2. 客户管理模块

### 2.1 客户列表页面

**路由**: `/customer/list`  
**布局**: 经典CRUD布局

#### 页面结构
```
┌──────────────────────────────────────────────────────┐
│ 快捷筛选标签                                          │
│ [全部] [我的客户] [今日需跟进] [即将过期] [公海客户]   │
├──────────────────────────────────────────────────────┤
│ 高级搜索(可折叠)                                      │
│ 公司名称:[______] 国家:[▼] 等级:[▼] 来源:[▼] [🔍]    │
├──────────────────────────────────────────────────────┤
│ 工具栏                                                │
│ [+新增] [批量分配] [批量导出] [放入公海] [📊视图切换] │
├──────────────────────────────────────────────────────┤
│ 视图模式: [列表视图] [卡片视图] [地图视图]             │
├──────────────────────────────────────────────────────┤
│ 客户列表(根据视图模式切换)                             │
└──────────────────────────────────────────────────────┘
```

#### 三种视图模式

**1. 列表视图(Table View)**
- 标准表格,适合快速浏览和批量操作
- 可自定义列显示/隐藏
- 支持列宽拖拽

**2. 卡片视图(Card View)**
```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ ABC Company │ │ XYZ Corp    │ │ DEF Ltd     │
│ 🇺🇸 美国      │ │ 🇩🇪 德国      │ │ 🇯🇵 日本      │
│ A级客户      │ │ B级客户      │ │ C级客户      │
│ 最近跟进:    │ │ 最近跟进:    │ │ 最近跟进:    │
│ 2天前       │ │ 5天前       │ │ 10天前      │
│ [详情] [跟进]│ │ [详情] [跟进]│ │ [详情] [跟进]│
└─────────────┘ └─────────────┘ └─────────────┘
```
- 瀑布流布局,每行3-4个卡片
- 卡片悬停有阴影和上浮效果
- 显示关键信息: 公司名、国旗、等级、跟进时间

**3. 地图视图(Map View)**
- 集成高德地图API
- 客户位置以标记点显示
- 不同颜色表示不同等级(A级红色,B级蓝色...)
- 点击标记点弹出客户信息气泡
- 支持按区域筛选

#### 交互细节
- **双击行**: 打开客户详情页
- **右键菜单**: 快速操作(编辑、删除、分配、跟进)
- **悬停提示**: 鼠标悬停在客户名称上,显示完整信息Tooltip

### 2.3 公海池页面

**路由**: `/customer/highsea`  
**特色**: 强调紧迫感和竞争感

#### 视觉设计
- 顶部统计卡片: "当前公海客户: 156个" (大号数字,醒目)
- 列表中的客户卡片显示"领取倒计时"(如刚释放的客户)
- "领取"按钮设计成醒目的橙色,悬停变大

#### 交互
- **领取确认**: 点击领取,弹窗确认"确定领取客户 XXX 吗?领取后将成为您的私海客户"
- **批量领取**: 最多一次领取10个,防止恶意刷取
- **冷却提示**: 刚释放的客户显示"XX分钟后可领取"

### 2.4 客户导入页面

**路由**: `/customer/import`  
**布局**: 步骤条引导(Stepper)

#### 步骤流程
```
① 下载模板 → ② 上传文件 → ③ 数据校验 → ④ 导入结果
```

**Step 1: 下载模板**
- 提供标准Excel模板下载
- 模板包含示例数据和字段说明

**Step 2: 上传文件**
- 拖拽上传区域,大图标+文字提示
- 支持.xlsx/.xls格式
- 显示文件大小限制(最大10MB)

**Step 3: 数据校验**
- 解析Excel,显示预览表格(前10条)
- 自动检测错误: 必填项缺失、格式错误、重复数据
- 错误行标红,鼠标悬停显示错误原因

**Step 4: 导入结果**
- 进度条显示导入进度
- 完成后显示: 成功XX条,失败XX条
- 可下载失败记录Excel(标注错误原因)

## 🎨 通用交互规范

### 加载状态
- 表格加载: 骨架屏(Skeleton)
- 按钮加载: `loading`属性,显示旋转图标
- 页面加载: 顶部进度条(NProgress)

### 空状态
- 列表为空: 显示插图 + "暂无数据" + [新增]按钮
- 搜索无结果: "未找到相关数据,试试其他关键词?"

### 错误提示
- 表单校验: 字段下方红色文字
- 接口错误: ElMessage.error('错误信息')
- 网络异常: 全局拦截,显示"网络连接失败,请检查网络"

### 成功反馈
- 保存成功: ElMessage.success('保存成功')
- 删除成功: ElMessage.success('删除成功')
- 重要操作: ElNotification通知(右上角,停留5秒)
