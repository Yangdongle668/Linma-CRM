## 🏗️ 技术架构亮点

### 1. 分层单体架构(可演进微服务)

```
前端展示层 (Vue 3)
    ↓
API网关层 (Spring Cloud Gateway - 预留)
    ↓
应用服务层 (Spring Boot 3.2)
    ↓
数据访问层 (MyBatis Plus 3.5)
    ↓
数据存储层 (MySQL + Redis + ES + MinIO)
```

## 📝 数据库设计亮点

### 1. 规范的表命名
- `sys_` 前缀: 系统管理表
- `crm_` 前缀: 业务数据表
- `msg_` 前缀: 消息通知表

### 2. 完善的索引设计
- 主键索引: BIGINT AUTO_INCREMENT
- 外键索引: customer_id, order_id等
- 查询索引: status, created_time等
- 联合索引: 多条件查询优化
- 全文索引: 公司名称搜索

### 3. 逻辑删除
- 所有表包含 `deleted` 字段
- MyBatis Plus自动处理

### 4. 审计字段
- created_by, created_time
- updated_by, updated_time

### 5. JSON字段
- attachment_urls (附件列表)
- photos (照片列表)
- tracking_data (物流轨迹)

## 🔐 安全设计

✅ **JWT认证**: Token + Refresh Token  
✅ **RBAC权限**: 角色-菜单-按钮三级权限  
✅ **数据权限**: 5种数据范围(全部/本部门/本部门及以下/仅本人/自定义)  
✅ **密码加密**: BCrypt强哈希  
✅ **操作审计**: AOP记录所有增删改操作  
✅ **敏感数据脱敏**: 手机号、邮箱脱敏显示  
✅ **接口限流**: IP级别、用户级别限流(预留)  
✅ **防SQL注入**: MyBatis预编译  
✅ **XSS防护**: 输入过滤(预留)
